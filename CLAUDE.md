## Project-Specific Claude Code Rules

Before working on this project, also read `PROJECT_RULES.md`.

- Write program code together with Haruta, in small understandable steps.
- Show proposed code or changes in the chat and explain them before editing
  files.
- Directories, documentation, plans, prompts, templates, and empty program
  files may be automatically created or edited without separate approval.
- For this non-program work, proceed directly without presenting numbered
  choices or requesting approval for each edit.
- Read-only research, web searches, and official documentation checks do not
  require Haruta's permission.
- Do not write program code into files or modify existing program code until
  Haruta understands and approves the proposed code.
- Do not present a complete implementation all at once when the goal is
  learning.
- The current camera module is OV3660, not OV2640.
